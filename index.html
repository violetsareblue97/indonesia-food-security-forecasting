01. Isu
Pengelolaan Stok Pangan Indonesia yang Kurang Proaktif
Ketergantungan pada metode konvensional dalam manajemen stok pangan nasional membuat antisipasi kebutuhan di masa depan menjadi tidak akurat. Hal ini mengakibatkan ketidakseimbangan pasokan antarwilayah, di mana surplus pangan sering kali tidak tersalurkan ke daerah yang justru mengalami kelangkaan.
02. Solusi
Solusi Menggunakan Strategi yang Proaktif
Selama ini, kebijakan pangan terjebak dalam pola reaktif, yaitu hanya merespons saat kelangkaan atau lonjakan harga sudah terjadi. Proyek ini memutar balik situasi tersebut, dengan menghadirkan sistem yang proaktif. Dengan prediksi akurat, pemerintah kini bisa bertindak lebih awal dan mengamankan stok pangan jauh sebelum krisis melanda.
03. Arsitektur Model
PatchTST untuk Data Deret Waktu Pendek
PatchTST membagi deret waktu menjadi segmen (patches) sebagai token Transformer. Dengan RevIN normalization dan channel-independent design, model bekerja efektif bahkan pada data historis yang sangat terbatas. Dataset yang digunakan pada project ini adalah data pangan nasional 2021-2023, dengan model ini hasil prediksi tetap memiliki akurasi yang tinggi

Teknologi

Tech Stack

Python
Bahasa utama untuk seluruh alur — ingest data, EDA, pelatihan model, hingga deployment.
PatchTST
Transformer berbasis patch untuk deret waktu. Tokenisasi subseries dengan mekanisme self-attention dan RevIN normalization.
NeuralForecast
Framework Nixtla untuk neural time series. Menangani multi-series training, format data, dan frekuensi tahunan.
PyTorch Lightning
Berjalan otomatis sebagai backend NeuralForecast. Mengatur training loop, logging MAE, dan eksekusi di CPU/GPU.
Pandas
Manipulasi data: groupby, agregasi, rename kolom, konversi tipe data, dan format ulang untuk NeuralForecast.
NumPy
Operasi numerik array, digunakan bersama Pandas untuk transformasi dan kalkulasi data sebelum masuk model.
Scikit-learn
StandardScaler untuk normalisasi variabel target — memastikan skala data stabil sebelum dan sesudah training.
Seaborn / Matplotlib
Visualisasi EDA: grafik tren konsumsi tahunan, bar chart komoditas tertinggi, dan profil distribusi data.
